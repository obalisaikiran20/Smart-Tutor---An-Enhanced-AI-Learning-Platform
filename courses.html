HTML Modules

HTML Structure

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design web pages. It provides the structure for a webpage by using tags and elements to define different parts of the content.

Basic Structure of an HTML Document

Every HTML document follows a standard structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Welcome to HTML!</h1>
    <p>This is my first webpage using HTML.</p>
  </body>
</html>
HTML Elements

What are HTML Elements?

HTML elements are the building blocks of a webpage. An element consists of an opening tag, content, and a closing tag.

Example of an HTML Element:

<p>This is a paragraph.</p>

Some commonly used HTML elements:

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Link
  • <img> - Image
  • <ul> & <li> - Unordered List
  • <ol> & <li> - Ordered List
  • <table> - Table
  • <form> - Form

Example of multiple elements in an HTML document:

<!DOCTYPE html>
<html>
  <head>
    <title>HTML Elements</title>
  </head>
  <body>
    <h1>Welcome to HTML!</h1>
    <p>This is a paragraph of text.</p>
    <a href="https://www.example.com">Click here</a>
    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
    </ul>
  </body>
</html>
HTML Attributes

What are HTML Attributes?

HTML attributes provide additional information about an element. They are always included inside the opening tag.

Example of an HTML Attribute:

<a href="https://www.example.com">Visit Example</a>

Common HTML Attributes:

  • href - Specifies the URL for links (<a> tag)
  • src - Specifies the image source (<img> tag)
  • alt - Provides alternative text for images
  • style - Adds inline CSS styling
  • id - Assigns a unique ID to an element
  • class - Assigns one or more class names

Example of Multiple Attributes:

<img src="image.jpg" alt="Sample Image" width="300" height="200">
HTML Headings

What are HTML Headings?

HTML headings are used to define titles and subtitles in a webpage. They range from <h1> (largest) to <h6> (smallest).

Example of HTML Headings:

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
<h4>This is Heading 4</h4>
<h5>This is Heading 5</h5>
<h6>This is Heading 6</h6>

Usage of Headings:

  • <h1> - Used for main titles (only one per page).
  • <h2> to <h6> - Used for subheadings and sections.

Example in a Webpage:

<html>
  <head>
    <title>HTML Headings Example</title>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <h2>About Us</h2>
    <h3>Our Services</h3>
  </body>
</html>
HTML Paragraphs

What are HTML Paragraphs?

HTML paragraphs are used to define blocks of text. The <p> tag is used to create paragraphs.

Example of an HTML Paragraph:

<p>This is a paragraph in HTML.</p>

Multiple Paragraphs:

<p>This is the first paragraph.</p>
<p>This is the second paragraph.</p>

Line Breaks: If you need to add a line break without starting a new paragraph, use the <br> tag.

<p>This is a line of text.<br>This text appears on a new line.</p>

Example in a Webpage:

<html>
  <head>
    <title>HTML Paragraphs Example</title>
  </head>
  <body>
    <p>Welcome to my webpage.</p>
    <p>HTML makes structuring content easy!</p>
  </body>
</html>
HTML Styles

What are HTML Styles?

HTML styles are used to change the appearance of elements. The style attribute is used to apply inline CSS to an element.

Example of Inline Styling:

<p style="color:blue; font-size:20px;">This is a styled paragraph.</p>

Common CSS Properties Used in HTML Styles:

  • color - Changes text color
  • font-size - Adjusts the font size
  • background-color - Sets the background color
  • text-align - Aligns text (left, center, right)
  • border - Adds borders to elements

Example of Multiple Styles Applied:

<p style="color:white; background-color:black; padding:10px; border-radius:5px;"> This is a paragraph with multiple styles. </p>

Example in a Webpage:

<html>
  <head>
    <title>HTML Styles Example</title>
  </head>
  <body>
    <p style="color:green; font-size:18px;"> Styling makes HTML pages visually appealing! </p>
  </body>
</html>
HTML Formatting

What is HTML Formatting?

HTML formatting is used to define the appearance of text, such as making it bold, italic, underlined, etc.

Common HTML Formatting Tags:

  • <b> - Bold text
  • <strong> - Important (bold) text
  • <i> - Italic text
  • <em> - Emphasized (italic) text
  • <mark> - Marked (highlighted) text
  • <small> - Smaller text
  • <del> - Deleted (strikethrough) text
  • <ins> - Inserted (underlined) text
  • <sub> - Subscript text
  • <sup> - Superscript text

Example of Formatting Tags:

<p>This is <b>bold</b>, <i>italic</i>, and <u>underlined</u> text.</p>

Example in a Webpage:

<html>
  <head>
    <title>HTML Formatting Example</title>
  </head>
  <body>
    <p>This is <strong>important</strong> text.</p>
    <p>This is <mark>highlighted</mark> text.</p>
    <p>This is <sub>subscript</sub> and <sup>superscript</sup> text.</p>
  </body>
</html>
HTML Quotations

What are HTML Quotations?

HTML provides several elements to define different types of quotations and citations.

Common HTML Quotation Tags:

  • <blockquote> - Defines a section quoted from another source
  • <q> - Defines a short, inline quotation
  • <cite> - Defines the title of a work (book, website, article, etc.)
  • <abbr> - Defines an abbreviation or acronym
  • <address> - Defines contact information

Example of Blockquote:

<blockquote>
  "The greatest glory in living lies not in never falling, but in rising every time we fall." - Nelson Mandela
</blockquote>

Example of Inline Quotation:

<p>According to Einstein, <q>Imagination is more important than knowledge.</q></p>

Example of Cite Tag:

<p><cite>The Great Gatsby</cite> is a novel written by F. Scott Fitzgerald.</p>

Example of Abbreviation Tag:

<p><abbr title="World Health Organization">WHO</abbr> is an international organization.</p>

Example of Address Tag:

<address>
  Written by John Doe.<br>
  Visit us at: example.com<br>
  Email: contact@example.com<br>
</address>
HTML Comments

What are HTML Comments?

HTML comments are used to add notes or explanations within the code. These comments are ignored by the browser and do not appear on the webpage.

Syntax of an HTML Comment:

<!-- This is a comment -->

Why Use Comments?

  • To explain the code for better readability
  • To temporarily disable parts of the code
  • To add reminders or notes for future reference

Example of Using Comments:

<!DOCTYPE html>
<html>
  <head>
    <title>HTML Comments Example</title>
  </head>
  <body>
    <!-- This is a heading -->
    <h1>Welcome to HTML Comments</h1>
    <!-- This paragraph will be displayed on the page -->
    <p>This is a sample paragraph.</p>
    <!-- <p>This paragraph is commented out and will not appear.</p> -->
  </body>
</html>
HTML Colors

What are HTML Colors?

HTML colors are used to style elements on a webpage. Colors can be applied using names, HEX codes, RGB, or HSL values.

Using Color Names:

<p style="color:red;">This is a red text</p>

Using HEX Codes:

<p style="color:#ff5733;">This is an orange text</p>

Using RGB Values:

<p style="color:rgb(0, 128, 0);">This is a green text</p>

Using HSL Values:

<p style="color:hsl(240, 100%, 50%);">This is a blue text</p>

Background Colors:

<div style="background-color:yellow; padding:10px;">This div has a yellow background</div>

Border Colors:

<div style="border:2px solid red; padding:10px;">This div has a red border</div>
HTML CSS

What is HTML CSS?

CSS (Cascading Style Sheets) is used to style HTML elements. It controls the layout, colors, fonts, and overall appearance of a webpage.

Ways to Apply CSS:

  • Inline CSS - Applied directly to an element using the style attribute.
  • Internal CSS - Written inside a <style> tag in the HTML document.
  • External CSS - Linked as a separate CSS file using the <link> tag.

Example of Inline CSS:

<p style="color:blue; font-size:20px;">This is a styled paragraph</p>

Example of Internal CSS:

<style>
  p { color: green; font-size: 18px; }
</style>
<p>This is a paragraph with internal CSS.</p>

Example of External CSS:

<link rel="stylesheet" href="styles.css">
<p>This is a paragraph styled with an external CSS file.</p>

CSS Selectors:

  • Element Selector - Targets all elements of a specific type (e.g., p { color: red; }).
  • Class Selector - Targets elements with a specific class (.class-name { color: blue; }).
  • ID Selector - Targets an element with a specific ID (#id-name { font-size: 20px; }).
HTML Links

What are HTML Links?

HTML links (or hyperlinks) allow users to navigate from one page to another. They are created using the <a> tag.

Basic Syntax of a Link:

<a href="https://www.example.com">Visit Example</a>

Common Link Attributes:

  • href - Specifies the URL of the page to navigate to.
  • target - Defines how the link will open (_blank for a new tab).
  • title - Provides additional information when hovering over the link.

Example with Different Attributes:

<a href="https://www.example.com" target="_blank" title="Go to Example">Click Here</a>

Linking to an Email Address:

<a href="mailto:someone@example.com">Send an Email</a>

Creating a Button Link:

<a href="https://www.example.com" style="display:inline-block; padding:10px 20px; background:#007bff; color:white; text-decoration:none; border-radius:5px;">Visit Example</a>
HTML Images

What are HTML Images?

HTML allows you to display images using the <img> tag. Images make web pages more visually appealing.

Basic Syntax of an Image:

<img src="image.jpg" alt="A sample image">

Common Image Attributes:

  • src - Specifies the image path (URL or local file).
  • alt - Provides alternative text if the image cannot be displayed.
  • width & height - Define image dimensions in pixels.
  • title - Shows a tooltip when hovering over the image.

Example with Attributes:

<img src="image.jpg" alt="Beautiful Scenery" width="400" height="300" title="Scenic View">

Using an Image as a Link:

<a href="https://www.example.com"> <img src="logo.png" alt="Example Logo" width="150"> </a>

Responsive Images (Adjusts Based on Screen Size):

<img src="image.jpg" alt="Responsive Image" style="max-width:100%; height:auto;">
HTML Tables

What are HTML Tables?

HTML tables are used to display data in rows and columns. The <table> tag is used to create a table.

Basic Table Structure:

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Alice</td>
    <td>22</td>
  </tr>
</table>

Table Elements:

  • <table> - Defines the table.
  • <tr> - Defines a row.
  • <th> - Defines a header cell (bold & centered by default).
  • <td> - Defines a standard cell.

Adding Borders and Styling:

<table style="border-collapse: collapse; width: 100%;" border="1">
  <tr>
    <th style="background-color: lightgray; padding: 10px;">Product</th>
    <th style="background-color: lightgray; padding: 10px;">Price</th>
  </tr>
  <tr>
    <td style="padding: 10px;">Laptop</td>
    <td style="padding: 10px;">$1000</td>
  </tr>
  <tr>
    <td style="padding: 10px;">Phone</td>
    <td style="padding: 10px;">$500</td>
  </tr>
</table>

Merging Cells (Colspan & Rowspan):

<table border="1">
  <tr>
    <th colspan="2">Merged Header</th>
  </tr>
  <tr>
    <td rowspan="2">Merged Cell</td>
    <td>Row 1</td>
  </tr>
  <tr>
    <td>Row 2</td>
  </tr>
</table>
HTML Lists

What are HTML Lists?

HTML lists allow you to organize and display information in an orderly manner. There are three types of lists in HTML:

  • Ordered List (<ol>) - A numbered list.
  • Unordered List (<ul>) - A bullet-point list.
  • Description List (<dl>) - A list of terms with descriptions.

Ordered List Example:

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

Unordered List Example:

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Orange</li>
</ul>

Description List Example:

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

Customizing List Styles:

<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

You can also use list-style-type with values like circle, disc, or none to modify list appearance.

HTML Blocks, IDs, and Classes

HTML Block and Inline Elements

HTML elements are categorized into two types:

  • Block Elements - These take up the full width available and start on a new line.
  • Inline Elements - These only take up as much width as necessary and stay in the same line.

Common Block Elements:

  • <div>
  • <p>
  • <h1> to <h6>
  • <ul>, <ol>
  • <table>

Example of Block Elements:

<div>This is a block element</div>
<p>This is a paragraph (also a block element)</p>

Common Inline Elements:

  • <span>
  • <a>
  • <strong>
  • <em>

Example of Inline Elements:

<p>This is a <span>highlighted text</span> inside a paragraph.</p>

HTML IDs

The id attribute gives a unique identifier to an element.

<p id="uniquePara">This paragraph has an ID.</p>

You can use CSS to style it:

#uniquePara { color: blue; font-weight: bold; }

HTML Classes

The class attribute is used to define a group of elements with a common styling.

<p class="highlight">This paragraph has a class.</p>
<p class="highlight">This is another paragraph with the same class.</p>

Using CSS, you can apply styles to all elements with the same class:

.highlight { color: red; font-size: 18px; }
HTML Audio

What is HTML Audio?

The <audio> element is used to embed sound content in a web page, such as music or audio clips.

Basic Example of an Audio Element:

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Attributes of the <audio> Element:

  • controls - Displays play, pause, and volume buttons.
  • autoplay - Automatically starts playing the audio.
  • loop - Repeats the audio after it ends.
  • muted - Starts the audio in a muted state.

Example with Multiple Formats (for Browser Compatibility):

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

Example with Autoplay and Loop:

<audio controls autoplay loop>
  <source src="background-music.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>
HTML Video

What is HTML Video?

The <video> element is used to embed video content in a web page, such as movies or video clips.

Basic Example of a Video Element:

<video controls>
  <source src="video.mp4" type="video/mp4">
  Your browser does not support the video element.
</video>

Attributes of the <video> Element:

  • controls - Displays play, pause, volume, and fullscreen buttons.
  • autoplay - Automatically starts playing the video.
  • loop - Repeats the video after it ends.
  • muted - Starts the video in a muted state.
  • poster - Displays an image before the video starts playing.

Example with Multiple Formats (for Browser Compatibility):

<video controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.ogv" type="video/ogg">
  <source src="video.webm" type="video/webm">
  Your browser does not support the video element.
</video>

Example with Autoplay, Loop, and Muted:

<video controls autoplay loop muted>
  <source src="background-video.mp4" type="video/mp4">
  Your browser does not support the video element.
</video>
HTML Forms

What is an HTML Form?

The <form> element is used to collect user input. It can contain various input elements like text fields, checkboxes, radio buttons, submit buttons, and more.

Basic Example of a Form:

<form action="submit_form.php" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

Form Attributes:

  • action - Specifies where to send the form data (e.g., a server-side script or URL).
  • method - Defines how the form data will be sent (e.g., GET or POST).

Input Types in Forms:

  • text - For a single-line text input.
  • password - For entering a password (input is hidden).
  • email - For entering an email address.
  • radio - For selecting a single option from a group.
  • checkbox - For selecting one or more options.
  • submit - A button to submit the form.

Example with Multiple Input Types:

<form action="submit_form.php" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  <label>Gender:</label>
  <input type="radio" id="male" name="gender" value="male"> Male
  <input type="radio" id="female" name="gender" value="female"> Female
  <label for="subscribe">Subscribe to newsletter:</label>
  <input type="checkbox" id="subscribe" name="subscribe">
  <input type="submit" value="Submit">
</form>

Form Validation:

HTML5 provides built-in form validation for required fields, email format, and more. You can also use the required, pattern, and minlength attributes to ensure the correct input.

Example with Required and Email Validation:

<form action="submit_form.php" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>
  <input type="submit" value="Submit">
</form>
