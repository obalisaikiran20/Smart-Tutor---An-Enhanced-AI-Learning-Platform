JavaScript Basics

What is JavaScript?

JavaScript is a programming language used to create dynamic and interactive effects within web browsers. It can modify HTML and CSS, handle user interactions, and much more.

Basic JavaScript Syntax:

A JavaScript statement is a single line of code that performs a specific task. Statements are often followed by semicolons.

var variableName = value;
// This is a single-line comment

Example:

var name = "John";
var age = 30;
console.log(name + " is " + age + " years old.");

Variables in JavaScript:

JavaScript uses variables to store data values. Variables can be declared using var, let, or const.

  • var: Used for variable declaration, but its scope is function-wide or global.
  • let: Block-scoped variable declaration.
  • const: A constant value that cannot be reassigned.

Example of Declaring Variables:

var x = 10;
let y = 20;
const z = 30;

JavaScript Functions:

Functions allow you to encapsulate code that can be reused multiple times. You can pass arguments to functions, and they can return values.

Example of a Function:

function greet(name) {
  console.log("Hello, " + name);
}
greet("John");

JavaScript Events:

JavaScript can be used to handle user events, such as clicking a button or submitting a form.

Example of an Event Listener:

document.getElementById("myButton").addEventListener("click", function() {
  alert("Button clicked!");
});

JavaScript Comments:

You can add comments to your JavaScript code using // for single-line comments or /* */ for multi-line comments.

// This is a single-line comment
/* This is a multi-line comment */
JavaScript Basics, Variables & Data Types

What is JavaScript?

JavaScript is a programming language that enables interactive web pages. It is one of the core technologies of web development alongside HTML and CSS. JavaScript is used for client-side and server-side development.

Basic JavaScript Syntax:

A JavaScript statement is a single line of code that performs a specific task. Each statement is generally terminated with a semicolon.

var variableName = value;
// Single-line comment

Example:

var name = "John";
var age = 30;
console.log(name + " is " + age + " years old.");

Variables in JavaScript:

Variables store values that can be used throughout the code. You can declare variables using var, let, or const.

  • var: Used to declare a variable with function-level scope. It is less commonly used in modern JavaScript.
  • let: Used to declare variables with block-level scope. It is commonly used in modern JavaScript development.
  • const: Used to declare variables whose value cannot be reassigned.

Example of Declaring Variables:

var x = 10;
let y = 20;
const z = 30;

Data Types in JavaScript:

JavaScript has various data types, including primitive and non-primitive types.

  • String: Used for textual data. Strings are enclosed in single or double quotes.
  • Number: Represents numeric values, both integers and floating-point numbers.
  • Boolean: Represents either true or false.
  • Null: Represents an intentional absence of any value.
  • Undefined: Represents a variable that has been declared but not assigned any value.
  • Object: Used to store collections of data, such as key-value pairs.
  • Array: A special type of object used to store ordered collections of values.

Examples of Data Types:

var name = "John"; // String
var age = 30; // Number
var isAdult = true; // Boolean
var address = null; // Null
var job; // Undefined
var person = { name: "John", age: 30 }; // Object
var colors = ["red", "green", "blue"]; // Array

Type Conversion in JavaScript:

JavaScript automatically converts types when necessary, but you can also manually convert between types.

Example of Type Conversion:

var strNumber = "123";
var number = Number(strNumber); // Converts string to number
console.log(number); // Output: 123 (as a number)

Check Data Type:

To check the data type of a value, you can use the typeof operator.

var name = "John";
console.log(typeof name); // Output: string
JavaScript Operators

What are JavaScript Operators?

Operators are used to perform operations on variables and values. JavaScript has various types of operators, such as arithmetic, comparison, logical, assignment, and more.

1. Arithmetic Operators:

Arithmetic operators are used to perform basic mathematical operations like addition, subtraction, multiplication, division, and more.

  • Addition (+): Adds two operands.
  • Subtraction (-): Subtracts the second operand from the first.
  • Multiplication (*): Multiplies two operands.
  • Division (/): Divides the first operand by the second.
  • Modulus (%): Returns the remainder of the division.
  • Exponentiation (**): Raises the first operand to the power of the second.
  • Increment (++): Increases a number by one.
  • Decrement (--): Decreases a number by one.

Examples:

var a = 10;
var b = 5;
var sum = a + b; // Addition
var difference = a - b; // Subtraction
var product = a * b; // Multiplication
var quotient = a / b; // Division
var remainder = a % b; // Modulus
var power = a ** b; // Exponentiation
a++; // Increment
b--; // Decrement

2. Assignment Operators:

Assignment operators are used to assign values to variables.

  • = Assigns the value of the right-hand operand to the left-hand operand.
  • += Adds the right-hand operand to the left-hand operand and assigns the result to the left operand.
  • -= Subtracts the right-hand operand from the left-hand operand and assigns the result to the left operand.
  • *= Multiplies the left-hand operand by the right-hand operand and assigns the result to the left operand.
  • /= Divides the left-hand operand by the right-hand operand and assigns the result to the left operand.
  • %= Assigns the remainder of the division to the left operand.
  • **= Assigns the result of the exponentiation to the left operand.

Examples:

var x = 10;
x += 5; // x = x + 5
x -= 3; // x = x - 3
x *= 2; // x = x * 2
x /= 4; // x = x / 4
x %= 3; // x = x % 3
x **= 2; // x = x ** 2

3. Comparison Operators:

Comparison operators are used to compare two values and return a boolean value (true or false).

  • == Equal to (compares only values).
  • === Strictly equal to (compares both value and type).
  • != Not equal to (compares only values).
  • !== Strictly not equal to (compares both value and type).
  • > Greater than.
  • < Less than.
  • >= Greater than or equal to.
  • <= Less than or equal to.

Examples:

var a = 5;
var b = 10;
console.log(a == b); // false
console.log(a !== b); // true
console.log(a > b); // false
console.log(a <= b); // true

4. Logical Operators:

Logical operators are used to combine multiple conditions.

  • && (AND): Returns true if both operands are true.
  • || (OR): Returns true if at least one operand is true.
  • ! (NOT): Reverses the logical state of its operand. Returns true if the operand is false, and vice versa.

Examples:

var x = true;
var y = false;
console.log(x && y); // false
console.log(x || y); // true
console.log(!x); // false

5. Conditional (Ternary) Operator:

The conditional operator (also known as the ternary operator) is a shorthand for the if-else statement. It takes three operands.

condition ? expr1 : expr2;

If the condition is true, expr1 is executed; otherwise, expr2 is executed.

Example:

var age = 18;
var canDrive = (age >= 18) ? "Yes" : "No"; // "Yes"
console.log(canDrive); // Output: Yes
JavaScript Conditional Statements

What are JavaScript Conditional Statements?

Conditional statements are used to perform different actions based on different conditions. They help in making decisions within the code.

1. if Statement:

The if statement is used to specify a block of code to be executed if a condition is true.

Syntax:

if (condition) {
   // Code to be executed if condition is true
}

Example:

var age = 18;
if (age >= 18) {
  console.log("You are an adult.");
}

2. else Statement:

The else statement is used to specify a block of code to be executed if the condition is false.

Syntax:

if (condition) {
   // Code to be executed if condition is true
} else {
   // Code to be executed if condition is false
}

Example:

var age = 16;
if (age >= 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}

3. else if Statement:

The else if statement is used when you want to specify a new condition to test, after the first condition is false.

Syntax:

if (condition1) {
   // Code to be executed if condition1 is true
} else if (condition2) {
   // Code to be executed if condition2 is true
} else {
   // Code to be executed if neither condition is true
}

Example:

var age = 20;
if (age >= 18) {
  console.log("You are an adult.");
} else if (age >= 13) {
  console.log("You are a teenager.");
} else {
  console.log("You are a child.");
}

4. switch Statement:

The switch statement is used to perform different actions based on different conditions. It’s often used when you have multiple conditions to test.

Syntax:

switch (expression) {
  case value1:
     // Code to be executed if expression === value1
    break;
  case value2:
     // Code to be executed if expression === value2
    break;
  default:
     // Code to be executed if expression doesn't match any case
}

Example:

var day = 3;
switch (day) {
  case 1:
    console.log("Monday");
    break;
  case 2:
    console.log("Tuesday");
    break;
  case 3:
    console.log("Wednesday");
    break;
  default:
    console.log("Invalid day");
}

5. ternary Operator (Conditional Operator):

The ternary operator is a shorthand for if-else and takes three operands: a condition, a result for true, and a result for false.

Syntax:

condition ? expr1 : expr2;

Example:

var age = 18;
var canVote = (age >= 18) ? "Yes" : "No";
console.log(canVote); // Output: Yes
JavaScript Loops

What are JavaScript Loops?

Loops are used to execute a block of code repeatedly as long as a specified condition is true. They help in automating repetitive tasks in programming.

1. for Loop:

The for loop is used when you know in advance how many times you want to execute a statement or a block of statements.

Syntax:

for (initialization; condition; increment/decrement) {
   // Code to be executed
}

Example:

for (var i = 0; i < 5; i++) {
  console.log(i);
}

This will output the numbers from 0 to 4.

2. while Loop:

The while loop is used to execute a block of code as long as the condition is true.

Syntax:

while (condition) {
   // Code to be executed
}

Example:

var i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

This will also output the numbers from 0 to 4.

3. do...while Loop:

The do...while loop is similar to the while loop, except that it guarantees the code block is executed at least once, even if the condition is false.

Syntax:

do {
   // Code to be executed
} while (condition);

Example:

var i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

This will also output the numbers from 0 to 4.

4. for...in Loop:

The for...in loop is used to loop through the properties of an object.

Syntax:

for (var key in object) {
   // Code to be executed
}

Example:

var person = {
  name: "John",
  age: 30,
  city: "New York"
};
for (var key in person) {
  console.log(key + ": " + person[key]);
}

This will output the properties and their values in the object.

5. for...of Loop:

The for...of loop is used to loop through the values of an iterable object (e.g., arrays).

Syntax:

for (var value of iterable) {
   // Code to be executed
}

Example:

var fruits = ["apple", "banana", "cherry"];
for (var fruit of fruits) {
  console.log(fruit);
}

This will output each fruit in the array.

JavaScript Exceptions and Debugging

What are JavaScript Exceptions?

Exceptions are runtime errors that occur when JavaScript code is executed. These errors may cause the script to stop running or behave unexpectedly. JavaScript provides mechanisms for handling errors and debugging code.

1. Types of JavaScript Exceptions:

  • Syntax Error: Occurs when the JavaScript code is incorrect or violates the language's syntax rules.
  • Reference Error: Occurs when a variable that has not been declared is referenced.
  • Type Error: Occurs when a value is not of the expected type.
  • Range Error: Occurs when a value is not within an allowable range.
  • Eval Error: Occurs when the eval() function is used incorrectly.

2. Handling Exceptions in JavaScript:

JavaScript provides try...catch blocks to handle exceptions. The code inside the try block is executed, and if an error occurs, the catch block is executed to handle the error.

Syntax:

try {
   // Code that may cause an exception
} catch (error) {
   // Code to handle the exception
}

Example:

try {
  var x = undefinedVariable;
} catch (error) {
  console.log("Error caught: " + error.message);
}

This will log: "Error caught: undefinedVariable is not defined".

3. Throwing Custom Errors:

You can throw your own exceptions using the throw statement. This is useful for validating inputs or creating custom error handling.

Syntax:

throw new Error("Custom error message");

Example:

function checkAge(age) {
  if (age < 18) {
    throw new Error("You must be 18 or older.");
  }
  console.log("Age is valid.");
}

try {
  checkAge(16);
} catch (error) {
  console.log("Error: " + error.message);
}

This will log: "Error: You must be 18 or older."

4. Debugging JavaScript:

Debugging is the process of finding and fixing errors or bugs in your code. JavaScript provides several tools for debugging:

  • Console Log: The console.log() method is commonly used to print values and check code flow.
  • Breakpoints: You can set breakpoints in the browser's developer tools to pause the execution and inspect the state of variables.
  • Debugger Statement: You can add the debugger; statement in your code, and the execution will pause at that point if developer tools are open.
  • Browser Developer Tools: Modern browsers (Chrome, Firefox, etc.) have built-in developer tools for inspecting variables, step-by-step execution, and network requests.

Example of Debugging with console.log:

function sum(a, b) {
  console.log("a:", a);
  console.log("b:", b);
  return a + b;
}

console.log(sum(3, 5));

This will print the values of a and b before returning their sum.

5. Using the Debugger Statement:

function calculate(a, b) {
  debugger;
  return a + b;
}

console.log(calculate(4, 6));

When the browser's developer tools are open, the script will pause at the debugger; statement, allowing you to inspect variables and control execution.

JavaScript Objects

What is a JavaScript Object?

A JavaScript object is a collection of key-value pairs. It is used to store data and more complex entities. In an object, each key (also known as a property) is paired with a value, and the values can be any data type (including other objects, arrays, or functions).

Creating an Object:

There are two primary ways to create objects in JavaScript:

  • Object Literal Syntax: The simplest way to create an object is by using curly braces {}.
  • Using the new Object() Syntax: You can also create objects by using the new Object() constructor, though it's less common.

1. Object Literal Syntax:

let person = {
  name: "John",
  age: 30,
  gender: "male"
};

This creates an object person with three properties: name, age, and gender.

2. Using the new Object() Syntax:

let person = new Object();
person.name = "John";
person.age = 30;
person.gender = "male";

Accessing Object Properties:

You can access an object’s properties using dot notation or bracket notation.

Dot Notation:

console.log(person.name); // John

Bracket Notation:

console.log(person["age"]); // 30

Updating Object Properties:

You can change the value of an existing property using either dot or bracket notation.

person.name = "Jane";
person["age"] = 25;

Adding New Properties to an Object:

You can add new properties to an object dynamically.

person.country = "USA";
person["city"] = "New York";

Deleting Object Properties:

You can delete properties from an object using the delete operator.

delete person.gender;

Example:

let car = {
  make: "Toyota",
  model: "Corolla",
  year: 2021,
  getCarInfo: function() {
    return this.make + " " + this.model + " " + this.year;
  }
};

console.log(car.getCarInfo()); // Toyota Corolla 2021

This example shows an object car with both data properties and a function (method) getCarInfo().

Object Methods:

Objects can also have methods, which are functions stored as properties of the object. You can define a method within an object to perform actions related to the object.

Example of a Method:

let person = {
  name: "Alice",
  greet: function() {
    console.log("Hello, " + this.name);
  }
};

person.greet(); // Hello, Alice

In this example, greet is a method that logs a message using the object's name property.

Iterating Over Object Properties:

You can loop through the properties of an object using the for...in loop.

for (let key in person) {
  console.log(key + ": " + person[key]);
}

Example of Iteration:

let car = {
  make: "Honda",
  model: "Civic",
  year: 2020
};

for (let key in car) {
  console.log(key + ": " + car[key]);
}

This will log all properties of the car object.

Nested Objects:

Objects can also contain other objects as values, allowing for a hierarchy of data.

let person = {
  name: "John",
  address: {
    street: "123 Main St",
    city: "New York"
  },
};

console.log(person.address.city); // New York
JavaScript Functions

What is a JavaScript Function?

A function is a reusable block of code designed to perform a particular task. Functions help in modularizing code and avoiding repetition.

Defining a Function:

Functions can be defined in JavaScript using the function keyword.

function greet() {
  console.log("Hello, world!");
}

Calling a Function:

greet(); // Outputs: Hello, world!

Function Parameters and Arguments:

Functions can accept parameters to make them more dynamic.

function greetUser(name) {
  console.log("Hello, " + name + "!");
}

greetUser("Alice"); // Outputs: Hello, Alice!

Returning Values from a Function:

function add(a, b) {
  return a + b;
}

let result = add(5, 3);
console.log(result); // Outputs: 8

Function Expressions:

let multiply = function(a, b) {
  return a * b;
};

console.log(multiply(4, 2)); // Outputs: 8

Arrow Functions (ES6):

let subtract = (a, b) => a - b;
console.log(subtract(10, 4)); // Outputs: 6

Function Scope:

Variables declared inside a function are only accessible within that function.

function testScope() {
  let message = "Inside function";
}

console.log(message); // Error: message is not defined

Default Parameters:

function greet(name = "Guest") {
  console.log("Hello, " + name);
}

greet(); // Outputs: Hello, Guest
JavaScript Arrays

What is a JavaScript Array?

An array is a special variable that can hold multiple values in a single reference. JavaScript arrays are dynamic, meaning they can grow and shrink as needed.

Creating an Array:

There are two ways to create an array in JavaScript:

  • Using Array Literal:
let fruits = ["Apple", "Banana", "Cherry"];
  • Using the new Array() Constructor:
let colors = new Array("Red", "Green", "Blue");

Accessing Array Elements:

console.log(fruits[0]); // Outputs: Apple

Modifying an Array:

fruits[1] = "Mango";
console.log(fruits); // ["Apple", "Mango", "Cherry"]

Array Properties:

console.log(fruits.length); // Outputs: 3

Common Array Methods:

  • push() - Adds an element to the end.
  • pop() - Removes the last element.
  • shift() - Removes the first element.
  • unshift() - Adds an element to the beginning.
fruits.push("Orange");
fruits.pop();
fruits.shift();
fruits.unshift("Pineapple");

Looping Through an Array:

fruits.forEach(function(fruit) {
  console.log(fruit);
});

Sorting and Reversing:

fruits.sort();
fruits.reverse();

Combining Arrays:

let veggies = ["Carrot", "Broccoli"];
let food = fruits.concat(veggies);

Splicing and Slicing:

fruits.splice(1, 1, "Strawberry");
let citrus = fruits.slice(1, 3);

Multidimensional Arrays:

let matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];
console.log(matrix[1][2]); // Outputs: 6
JavaScript Strings, Numbers, and Math

JavaScript Strings

Strings in JavaScript are sequences of characters used to store and manipulate text.

Creating Strings:

let str1 = "Hello"; // Double quotes
let str2 = 'World'; // Single quotes
let str3 = `Hello, ${str2}!`; // Template literals
console.log(str3); // Output: "Hello, World!"

Common String Methods:

let text = "JavaScript is awesome!";
console.log(text.length); // Get length: 22
console.log(text.toUpperCase()); // Convert to uppercase
console.log(text.toLowerCase()); // Convert to lowercase
console.log(text.indexOf("is")); // Find position of substring
console.log(text.replace("awesome", "powerful")); // Replace text
console.log(text.slice(0, 10)); // Extract substring

Concatenation:

let first = "Hello";
let second = "World";
let result = first + " " + second;
console.log(result); // Output: "Hello World"

JavaScript Numbers

Numbers in JavaScript can be integers or floating points.

Working with Numbers:

let x = 10;
let y = 20.5;
console.log(x + y); // Output: 30.5
console.log(typeof x); // Output: "number"

Number Methods:

let num = 123.456;
console.log(num.toFixed(2)); // Output: "123.46" (rounds to 2 decimal places)
console.log(num.toPrecision(4)); // Output: "123.5" (total 4 digits)
console.log(Number.isInteger(num)); // Output: false

JavaScript Math

The Math object provides mathematical functions and constants.

Common Math Methods:

console.log(Math.round(4.6)); // Output: 5
console.log(Math.floor(4.9)); // Output: 4 (rounds down)
console.log(Math.ceil(4.1)); // Output: 5 (rounds up)
console.log(Math.abs(-10)); // Output: 10 (absolute value)
console.log(Math.max(10, 20, 30)); // Output: 30 (max value)
console.log(Math.min(10, 20, 30)); // Output: 10 (min value)

Generating Random Numbers:

console.log(Math.random()); // Random number between 0 and 1
console.log(Math.floor(Math.random() * 10) + 1); // Random number between 1 and 10

Conclusion

JavaScript provides powerful string, number, and math functionalities for handling data efficiently.

JavaScript Map and Set

JavaScript Map

The Map object holds key-value pairs and remembers the original insertion order of keys.

Creating a Map:

let myMap = new Map();
myMap.set("name", "John");
myMap.set("age", 25);
myMap.set("city", "New York");
console.log(myMap); // Output: Map { "name" => "John", "age" => 25, "city" => "New York" }

Accessing Values:

console.log(myMap.get("name")); // Output: John
console.log(myMap.get("age")); // Output: 25

Checking Keys:

console.log(myMap.has("city")); // Output: true
console.log(myMap.has("country")); // Output: false

Deleting Elements:

myMap.delete("age");
console.log(myMap.has("age")); // Output: false

Iterating Over a Map:

myMap.forEach((value, key) => {
  console.log(`${key}: ${value}`);
});

Map Size:

console.log(myMap.size); // Output: 2

JavaScript Set

The Set object allows you to store unique values of any type.

Creating a Set:

let mySet = new Set();
mySet.add(1);
mySet.add(2);
mySet.add(3);
mySet.add(2); // Duplicate values are ignored
console.log(mySet); // Output: Set { 1, 2, 3 }

Checking for a Value:

console.log(mySet.has(2)); // Output: true
console.log(mySet.has(5)); // Output: false

Deleting Elements:

mySet.delete(3);
console.log(mySet.has(3)); // Output: false

Set Size:

console.log(mySet.size); // Output: 2

Iterating Over a Set:

mySet.forEach(value => {
  console.log(value);
});

Conclusion

Map: Stores key-value pairs and maintains insertion order.

Set: Stores unique values and prevents duplicates.
